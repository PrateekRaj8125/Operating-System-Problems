# 🤝 Contributing to Operating-System-Problems

Thank you for considering contributing! 🎉  
Your contributions help improve this repository as a learning resource for Operating Systems.

---

## 📌 Ways to Contribute

You can contribute in multiple ways:

- 🐞 Fix bugs or incorrect logic
- 💡 Suggest new OS problems or algorithms
- 📝 Improve documentation (README, comments)
- 💻 Add new programs (C / Shell scripting)
- ⚡ Optimize existing implementations

## 🛠 Development Setup

1. Fork the repository
2. Clone your fork:

    ```bash
        git clone https://github.com/PrateekRaj8125/Operating-System-Problems.git
        cd Operating-System-Problems
    ```

3. Run C Programs

    ```bash
        gcc program.c -o program
        ./program
    ```

4. Run Shell Scripts

    ```bash
        chmod +x script.sh
        ./script.sh
    ```

## ✅ Contribution Guidelines

- Follow clean coding practices:
  - Proper indentation
  - Meaningful variable names
  - Clear comments
- Ensure:
  - Code compiles without errors
  - Output is correct
  - Logic is efficient and readable

---

## 🐞 Reporting Issues

Use GitHub Issues and include:

- Description of the issue
- Steps to reproduce
- Expected vs actual output
- Screenshots/logs (if needed)

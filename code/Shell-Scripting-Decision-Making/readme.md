# Shell Scripting - Decision Making

This folder contains shell scripting programs focused on **decision-making and control structures** such as conditional statements and loops.

---

## 📚 Topics Covered

- Conditional statements (if-else, nested conditions)
- Logical operators
- Loop constructs (for loop)
- Number-based problem solving

## 🎯 Purpose

This collection helps in:

- Applying decision-making logic in shell scripts
- Understanding control flow using conditions and loops
- Solving mathematical problems using scripting

---

## 🚀 How to Run

1. Give execution permission:

    ```bash
        chmod +x filename.sh
    ```

2. Run the script:

    ```bash
        ./filename.sh
    ```

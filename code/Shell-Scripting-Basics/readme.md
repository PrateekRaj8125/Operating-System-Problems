# Shell Scripting Basics

This folder contains basic shell scripting programs designed to introduce fundamental concepts of **Linux shell scripting**.

---

## 📚 Topics Covered

- Basic shell script structure
- Input and output operations
- Conditional statements (if-else)
- Arithmetic operations

---

## 📂 Program List

| S. No. | Questions | File Name |
| --- | --- | --- |
| 1. | Write a shell script to print "Hello World". | [hello_world.sh]() |
| 2. | Write a shell script to accept and display user name. | [read_and_display_name.sh]() |
| 3. | Write a shell script to find sum of two numbers. | [sum_of_two_numbers.sh]() |
| 4. | Write a shell script to calculate grade based on percentage. | [grade_calculator.sh]() |

---

## 🎯 Purpose

This collection helps in:

- Understanding basic shell scripting syntax
- Learning user interaction in scripts
- Applying conditional logic
- Performing simple computations

---

## 🚀 How to Run

1. Give execution permission:

    ```bash
        chmod +x filename.sh
    ```

2. Run the script:

    ```bash
        ./filename.sh
    ```

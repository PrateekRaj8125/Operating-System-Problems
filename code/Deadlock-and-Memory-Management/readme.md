# Deadlock and Memory Management

This folder contains C programs related to **deadlock handling and memory management techniques** in Operating Systems.

---

## 📚 Topics Covered

- Banker’s Algorithm (Deadlock Avoidance)
- Memory Allocation Techniques
- Multiprogramming with Fixed Tasks (MFT)
- Multiprogramming with Variable Tasks (MVT)

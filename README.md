# Operating System Problems 🖥️

This repository contains a **comprehensive collection of Operating System programs**, systematically organized from **basic shell scripting to advanced OS algorithms**.

It is designed as a complete learning resource for Operating Systems, covering:

- Shell scripting fundamentals
- Process scheduling algorithms
- Deadlock handling
- Memory management techniques

## 🚀 How to Use

▶️ For C Programs

1. Compile:

    ```bash
        gcc filename.c -o output
    ```

2. Run:

    ```bash
        ./output
    ```

▶️ For Shell Scripts

1. Give permission:

    ```bash
        chmod +x filename.sh
    ```

2. Execute:

    ```bash
        ./filename.sh
    ```
